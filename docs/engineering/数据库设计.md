# 数据库设计

## ER图概览

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│    User     │────<│  Notebook   │────<│  Document   │
└─────────────┘     └─────────────┘     └─────────────┘
                           │                    │
                           │                    │
                           ▼                    ▼
                    ┌─────────────┐     ┌─────────────┐
                    │    Tag      │     │  Document   │
                    └─────────────┘     │   Chunk     │
                                       └─────────────┘
                           │
                           ▼
                    ┌─────────────┐
                    │Conversation │────<│  Message    │
                    └─────────────┘     └─────────────┘
```

## 表结构设计

### users (用户表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| username | VARCHAR(50) | 用户名 |
| email | VARCHAR(100) | 邮箱 |
| password_hash | VARCHAR(255) | 密码哈希 |
| avatar | VARCHAR(255) | 头像URL |
| is_active | BOOLEAN | 是否激活 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### notebooks (Notebook表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID (FK) |
| name | VARCHAR(100) | Notebook名称 |
| description | TEXT | 描述 |
| is_favorite | BOOLEAN | 是否收藏 |
| created_at | DATETIME | 创建时间 |
| updated_at | DATETIME | 更新时间 |

### documents (文档表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| notebook_id | BIGINT | Notebook ID (FK) |
| name | VARCHAR(255) | 文件名 |
| file_path | VARCHAR(500) | 存储路径 |
| file_size | BIGINT | 文件大小 |
| file_type | VARCHAR(20) | 文件类型 |
| status | ENUM | 状态 |
| chunk_count | INT | 分块数量 |
| created_at | DATETIME | 创建时间 |

### document_chunks (文档分块表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| document_id | BIGINT | 文档ID (FK) |
| content | TEXT | 文本内容 |
| metadata | JSON | 元数据(页码/章节) |
| vector_id | VARCHAR(100) | 向量ID |
| position | INT | 位置序号 |

### conversations (会话表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| notebook_id | BIGINT | Notebook ID (FK) |
| title | VARCHAR(100) | 会话标题 |
| created_at | DATETIME | 创建时间 |

### messages (消息表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| conversation_id | BIGINT | 会话ID (FK) |
| role | ENUM | 角色(user/assistant) |
| content | TEXT | 消息内容 |
| citations | JSON | 引用来源 |
| created_at | DATETIME | 创建时间 |

### tags (标签表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID (FK) |
| name | VARCHAR(50) | 标签名 |
| color | VARCHAR(7) | 颜色值 |

### favorites (收藏表)

| 字段 | 类型 | 说明 |
|------|------|------|
| id | BIGINT | 主键 |
| user_id | BIGINT | 用户ID (FK) |
| target_type | ENUM | 收藏类型 |
| target_id | BIGINT | 收藏对象ID |
| created_at | DATETIME | 创建时间 |

## 索引设计

```sql
-- notebooks
CREATE INDEX idx_notebooks_user ON notebooks(user_id);
CREATE INDEX idx_notebooks_favorite ON notebooks(is_favorite);

-- documents
CREATE INDEX idx_documents_notebook ON documents(notebook_id);
CREATE INDEX idx_documents_status ON documents(status);

-- document_chunks
CREATE INDEX idx_chunks_document ON document_chunks(document_id);

-- messages
CREATE INDEX idx_messages_conversation ON messages(conversation_id);
```
