## V1.0 - 基础版本

### 核心功能

- [x] 用户系统（注册/登录/JWT 认证/退出/Token 刷新）
- [x] Notebook 管理（CRUD/收藏/搜索）
- [x] 文档管理（上传/解析/状态追踪）
- [x] AI 聊天（会话/消息/历史记录）
