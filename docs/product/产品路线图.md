# 产品路线图

## V1.0 - 基础版本

### 核心功能

- [ ] 用户系统（注册/登录/JWT认证）
- [ ] Notebook管理（CRUD/收藏/搜索）
- [ ] 文档管理（上传/解析/状态追踪）
- [ ] AI聊天（多轮对话/上下文/历史记录）

| 阶段 | 目标 | 预计时间 |

| M1 | 项目初始化、基础框架 | Week 1-2 |
| M2 | 用户系统、Notebook | Week 3-4 |

| M4 | AI聊天、RAG | Week 7-8 |
| M5 | 测试、优化、发布 | Week 9-10 |
